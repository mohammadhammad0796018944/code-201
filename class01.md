# What are Semantic elements ? Semantic elements = elements with a meaning.

# this is exaple for Semantic elements.

1. <article> element specifies independent, self-contained content.
2. <aside> element defines some content aside from the content it is placed in (like a sidebar).
3. <details> tag Defines additional details that the user can view or hide
4. <header> element represents a container for introductory content or a set of navigational links.
5. <footer> element defines a footer for a document or section.
6. <nav> element defines a set of navigation links.
7. <figure> tag specifies self-contained content, like illustrations, diagrams, photos, code listings
8. <figcaption> tag defines a caption for a figure element. The figcaption element can be placed as the first or as the last child of a figure element.
9. <main> tag Specifies the main content of a document
10. <mark> tag Defines marked/highlighted text
11. <section> tag Defines a section in a document
12. <summary> tag Defines a visible heading for a details element
13. <time> tag Defines a date/time

# (Links to an external site.)What is a sitemap ?
A sitemap is a file where you provide information about the pages, videos, and other files on your site, and the relationships between them. Search engines like Google read this file to more intelligently crawl your site. A sitemap tells Google which pages and files you think are important in your site, and also provides valuable information about these files: for example, for pages, when the page was last updated, how often the page is changed, and any alternate language versions of a page.
